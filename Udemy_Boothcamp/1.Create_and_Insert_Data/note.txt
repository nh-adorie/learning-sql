** Mỗi khi vào cần chạy ở ternimal: sudo service mysql start // sudo mysql // exit

1. Creating database and table:

- CREATE DATABASE <dtb name>;
- SHOW DATABASES;
- DROP DATABASE <dtb name>;

- USE <database name>; // Chọn database mà mình sẽ sử dụng
- SELECT database(); // Kiểm tra lại xem database nào đang được sử dụng
Khi sử dụng trên giao diện MySQL Workbench, có thể double click vào tên database và ấn RUN
==> Khi đó database sẽ được chọn để sử dụng

// Database is made of a lot of tables

- CREATE TABLE <table name>
    (
        column_name dt_type,
        column_name dt_type,
    );

- SHOW TABLES;
- SHOW COLUMNS FROM <table_name>;
= DESC <table_name>;
= DESCRIBE <table_name>;
Khi sử dụng trên giao diện MySQL Workbench, thông tin này được hiển thị ở panel nhỏ bên cạnh rùi

- DROP TABLE <table_name>;

- INSERT INTO <table_name>(attr) VALUES(value);

- SELECT * FROM <table_name>;

* Phân loại câu lệnh trong SQL:
- DDL (Data Definition Language): định nghĩa cấu trúc dữ liệu → CREATE, ALTER, DROP, TRUNCATE…
- DML (Data Manipulation Language): thao tác với dữ liệu bên trong bảng → INSERT, UPDATE, DELETE…
- DQL (Data Query Language): truy vấn dữ liệu → SELECT 
- DCL (Data Control Language): phân quyền, quản lý user → GRANT, REVOKE…
